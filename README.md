# Solar System Simulation

A 3D solar system simulation built with Three.js featuring all planets orbiting the Sun.

## Features

- Complete solar system with Sun and all 9 planets (including Pluto)
- Realistic planet textures
- Moon orbiting Earth
- Orbit path visualizations
- Interactive camera controls (pan, zoom, rotate)
- Wireframe toggle button
- Smooth animations

## How to Run

1. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```

2. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

## Controls

- **Left Mouse Button + Drag**: Rotate camera
- **Middle Mouse Button + Drag**: Pan camera
- **Scroll Wheel**: Zoom in/out
- **Arrow Keys**: Pan camera
- **Toggle Wireframes Button**: Show/hide planet wireframes

## Planets

- Sun
- Mercury
- Venus
- Earth (with Moon)
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune
- Pluto
